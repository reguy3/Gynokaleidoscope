"_scope" was created in 48 hours as part of the Global Game Jam,
between Jan 25 2013 - Jan 27 2013, at The College of New Jersey in Ewing, NJ, United States.

"_scope" was created in Processing and compiled to Java. To play, open the "release" folder, and run the executable inside the folder of the platform of your choice.
Alternatively, you can download the Processing libraries and IDE free from http://processing.org, and compile from the src directory.
